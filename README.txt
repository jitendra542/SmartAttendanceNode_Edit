
Smart Attendance (Node + Express) - QR based demo
================================================

Requirements:
- Node.js (14+), npm
- (Optional) A modern browser with webcam for scanning

Install & Run:
1. Extract the zip
2. In terminal, change directory to project folder
   cd SmartAttendanceNode
3. Install dependencies:
   npm install
4. Start server:
   npm start
5. Open in browser:
   http://localhost:3000

Notes:
- QR images are saved in public/qrs/
- DB file is attendance.db (SQLite) in project root
